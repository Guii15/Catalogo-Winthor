# 🛒 Catálogo Digital de Produtos - Integração WinThor

Este projeto é um catálogo digital interativo desenvolvido para otimizar a visualização de produtos e a gestão de pedidos. Ele foi criado para facilitar a transição entre o estoque físico/ERP e a venda final.

## 🚀 Funcionalidades
* **Visualização Dinâmica:** Exibição de produtos com dados atualizados.
* **Filtros Inteligentes:** Busca rápida por código ou descrição.
* **Integração WhatsApp:** Botão de fechamento de pedido que envia os itens escolhidos direto para o vendedor.
* **Responsividade:** Interface adaptada para celulares e computadores.

## 🛠️ Tecnologias Utilizadas
* **JavaScript (ES6+):** Lógica de interface e manipulação de dados.
* **Vite:** Ferramenta de build para um desenvolvimento rápido e otimizado.
* **HTML5 & CSS3:** Estruturação e estilização moderna.
* **Integração de Dados:** Preparado para consumir dados originados do ERP WinThor.

## 📦 Como rodar o projeto
1. Clone o repositório: `git clone https://github.com/Guii15/catalogo-winthor.git`
2. Instale as dependências: `npm install`
3. Inicie o servidor local: `npm run dev`
